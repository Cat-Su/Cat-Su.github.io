Running android apps on Windows

Step 1: Enable Virtualization in BIOS

If your PC does not support virtualization, you won't be able to run Android apps on your PC.

Step 2: Enable Windows Features

Search for "Windows features" and open "Turn Windows features on or off" to ensure necessary components are enabled. Follow these steps:

  1. Press the Windows key on your keyboard.
  2. Type "Windows features" in the search bar.
  3. Select "Turn Windows features on or off" from the search results.
  4. In the Windows Features window, ensure that the following features are checked:
    • "Virtual Machine Platform"
    • Check the entire "Hyper-V" group, which may include multiple features related to Hyper-V.
  5. Click "OK".
  6. Restart your computer.

Step 3: Open Developer Settings

Accessing developer settings is crucial for optimizing your environment for Android app development. Follow these steps:

  1. Press the Windows key on your keyboard.
  2. Type "Developer settings" in the search bar.
  3. Select "Developer settings" from the search results.
  4. In the Developer settings window, you may find options to enable USB debugging, OEM unlocking, or other developer-related features. Enable these options if necessary.
  5. Turn on Developer mode, Enable the option to Install apps from any source or a similar setting depending on your Windows version..

Step 4: Download Windows Subsystem for Android

Download the Windows Subsystem for Android to enable seamless integration of Android apps on your Windows PC. Follow these steps:

  1. Visit the GitHub repository for Windows Subsystem for Android by going to https://github.com/MustardChef/WSABuilds.
  2. Click on the latest stable version for your operating system and architecture.
  3. Scroll down to find the “…release-nightly-MindTheGapps-XX.X-RemovedAmazon_…” version.
  4. Download the installation package (usually a ZIP file) to your computer.
  5. Extract the contents of the ZIP file to a folder on your computer.

Note: Select the latest stable version compatible with your operating system (e.g., Windows 10) and architecture (e.g., 64-bit).

Step 5: Extract Contents of the Downloaded File

After downloading the Windows Subsystem for Android, extract the contents of the .7z file using a tool like 7-Zip. Follow these steps:

  1. Download and install 7-Zip from https://7-zip.org if you don't have it installed.
  2. Open 7-Zip from your Start Menu or desktop.
  3. Navigate to the location where you downloaded the .7z file using the 7-Zip file manager.
  4. Click on the .7z file to select it.
  5. Press the Extract button on the menu bar.
  6. Choose or create a destination folder for the extraction.
  7. Click OK to start the extraction process.
  8. Wait for the extraction process to complete.

Step 6: Rename and Move the Extracted Folder

In the folder containing the extracted files, you'll find a folder. Rename it to "WSA" and move the renamed folder to the C:\ directory. Follow these steps:

  1. Open the folder where you extracted the contents of the .7z file.
  2. You'll find a folder within the extracted files (e.g., the main Windows Subsystem for Android folder).
  3. Right-click on the folder to select it.
  4. Choose the Rename option from the context menu.
  5. Type "WSA" as the new name and press Enter to confirm the change.
  6. Once renamed, Cut the folder (Ctrl+X).
  7. Navigate to the C:\ directory (you can open File Explorer and go to "This PC" > "Local Disk (C:)").
  8. Paste the folder into the C:\ directory (Ctrl+V).

Step 7: Run the Installation Script

In the "C:\WSA" directory, you'll find a file named "Run.bat." Double-click on it to run the installation script. Follow these steps:

  1. Navigate to the C:\WSA directory (you can open File Explorer and go to "This PC" > "Local Disk (C:)" > "WSA").
  2. You'll find a file named "Run.bat".
  3. Double-click on the "Run.bat" file to execute the installation script.
  4. Follow any on-screen prompts or instructions to complete the installation.

Step 8: Access Google Play Store and Log In

If Google Play Store hasn't launched already, follow these steps:

  1. Open Google Play Store from your Start menu.
  2. Log into your Google account within Google Play Store.
  3. After logging in, Google Play Store may crash. Don't worry, this is likely Google Play updating itself.
  4. Re-launch Google Play Store as many times as needed until it stabilizes.

Step 9: Install Apps from Google Play Store

You should now be able to install apps from the Google Play Store. Have fun exploring and enhancing your Windows Subsystem for Android experience!
